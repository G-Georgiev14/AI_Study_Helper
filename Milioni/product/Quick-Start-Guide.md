## 📱 Quick Reference Examples

### Essay Emergency? 📝
```
Act as an expert writing teacher. I need to write a 3-page essay about "The impact of social media on teenage mental health" for my 10th grade English class. Create a detailed outline with introduction, 3 body paragraphs, and conclusion. Include specific examples and counterarguments.
```

### Math Problem Stuck? 🔢
```
I'm stuck on this algebra problem: 2x + 5 = 3x - 7. Solve it step-by-step and explain each step like I'm 14 years old. Show me how to check my answer.
```

### Research Project? 📖
```
I'm doing a research project on renewable energy for 9th grade science. Help me develop 5 specific research questions that I can answer using online sources. Make sure they're not too broad or too narrow.
```

## 🆘 Troubleshooting

### AI Not Understanding?
- Try rephrasing your prompt
- Add more specific details
- Break big requests into smaller parts
- Give examples of what you want

### Not Getting Good Results?
- Check if your prompt is clear enough
- Make sure you provided all necessary context
- Try a different AI tool
- Ask AI to explain something differently
