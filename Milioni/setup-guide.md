## 🚀 Setting Up Your Sales Website (5 Minutes)

### Step 1: Update Payment Links

Open `index.html` and find the purchase buttons (around line 180):

```html
<button class="primary-btn" onclick="window.open('https://gumroad.com/l/codelaunch', '_blank')">
    Buy on Gumroad →
</button>
<button class="secondary-btn" onclick="window.open('https://payhip.com/b/codelaunch', '_blank')">
    Buy on Payhip →
</button>
```

Replace these URLs with your actual Gumroad/Payhip product links.

### Step 2: Customize Your Brand

Update the header section (line 12-14):

```html
<h1>CodeLaunch Portfolio</h1>
<p class="tagline">Your professional developer portfolio in 10 minutes</p>
```

### Step 3: Deploy Your Website

**Option A: GitHub Pages (Free)**
1. Create a new GitHub repository
2. Upload all files (except the product folder)
3. Go to Settings → Pages
4. Select main branch and save
5. Your site is live at `https://username.github.io/repository`

**Option B: Netlify (Free)**
1. Go to netlify.com
2. Drag and drop your files
3. Get instant deployment

## 🔧 Troubleshooting

### Common Issues

**Payment links not working:**
- Double-check your Gumroad/Payhip URLs
- Test the links in incognito mode

**Portfolio template not loading:**
- Check all file paths in the HTML
- Ensure CSS and JS files are linked correctly

**Mobile responsiveness issues:**
- Test on actual mobile devices
- Check viewport meta tag in HTML

### Getting Help

1. Check the README.md in the product folder
2. Review this setup guide
3. Test changes incrementally
4. Use browser developer tools for debugging
